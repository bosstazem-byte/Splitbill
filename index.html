ตั้งค่า Firebase
จำเป็นสำหรับ real-time sync

วิธีตั้งค่า

แอปนี้ใช้ Firebase Realtime Database สำหรับ sync แบบ real-time แก้ไขไฟล์ index.html ตามขั้นตอนด้านล่าง

1. ไปที่ console.firebase.google.com → สร้าง Project ใหม่
2. ไปที่ Build → Realtime Database → Create database → เลือก "Start in test mode"
3. ไปที่ Project Settings → เลื่อนลง → Add app (Web) → คัดลอก firebaseConfig
4. เปิดไฟล์ index.html → หา const firebaseConfig → แทนที่ค่า YOUR_... ทั้งหมด
5. Push ขึ้น GitHub → เปิด Pages → แชร์ URL ให้เพื่อน 🎉
